## 🎯 Overview

This project is designed to:
1. **Automate** data cleaning and preprocessing from admin Excel/Sheets files
2. **Analyze** company performance with comprehensive data analytics
3. **Deliver insights** for strategic decision making and business growth

By leveraging historical vacation registration data, this project identifies trends, patterns, and growth opportunities to improve business strategy.

---

## ✨ Key Features

### 1. **Excel & Google Sheets Data Automation & Cleaning**
- ✅ Data validation and quality checks from multiple sources
- ✅ Missing value handling and duplicate removal
- ✅ Data standardization and format normalization
- ✅ Automated data transformation pipelines
- ✅ Export-ready clean datasets

**Data Processed:**
- 👥 Participant biodata (name, age, contact information)
- 🎫 Selected vacation packages
- 📍 Pickup addresses and destinations
- 💰 Package prices and payment information
- 📅 Registration dates and booking details

### 2. **Company Performance Analytics**
- 📊 **Exploratory Data Analysis (EDA)**: Data distribution, summary statistics, quality assessment
- 📈 **Trend Analysis**: 
  - Monthly and yearly participant trends
  - Seasonal patterns in registrations
  - Year-over-year growth analysis
- 🎯 **Product Analytics**:
  - Participant count per vacation package
  - Product popularity ranking
  - Revenue contribution analysis
- 💹 **Revenue Analysis**:
  - Monthly and yearly revenue trends
  - Average revenue per participant
  - Growth rate calculations
- 🔮 **Forecasting**:
  - Participant predictions for next year
  - Revenue forecasting
- 🔗 **Statistical Analysis**:
  - Correlation analysis between variables
  - Regression analysis
  - Seasonality decomposition
